# Blog Title Clustering App üìù

A web application that scrapes blog titles from [Real Python](https://realpython.com/), clusters them using **KMeans**, and allows users to predict the cluster of a new blog title. Built with **Python**, **Streamlit**, and **scikit-learn**.

---

## Features

- Scrapes blog titles and links from Real Python.  
- Cleans and preprocesses text data.  
- Converts titles to numerical features using TF-IDF.  
- Clusters titles automatically using KMeans.  
- Provides an interactive Streamlit interface to:  
  - View scraped blog titles and their clusters.  
  - Predict the cluster of a new blog title.
